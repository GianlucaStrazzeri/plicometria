# Overview — plicometria-app

Este documento es la puerta de entrada a la documentación del proyecto y enlaza las secciones específicas:

- `frontend.md` — estructura y componentes UI del cliente.
- `backend.md` — API routes, Prisma y datos.
- `deployment.md` — recomendaciones para build y despliegue (Vercel, env vars).

Uso rápido:
- Lee `frontend.md` si trabajas en componentes o en la experiencia del usuario.
- Lee `backend.md` si trabajas con las API o la base de datos.
- Lee `deployment.md` antes de desplegar en entornos cloud.

---

Para cualquier cambio en la estructura del proyecto, actualiza también `PROJECT_STRUCTURE.md` en la raíz.
